Pwnshop mashup: foursquare-iens
====================

This is a codeigniter webapp which queries the foursquare venues api, integrates it with data from iens.nl and returns it on a google maps map.

It makes use of:
* Codeigniter for the MVC part
* jQuery
* Bootstrap
* phpQuery

* The foursquare API
* The iens.nl website
